# Endless Runner Game (Unity)

This project is a 3-lane endless runner game developed using Unity and C#.

## Features
- Lane switching (left - right movement)
- Jump and slide mechanics
- Obstacle spawning system
- Score system
- Increasing game speed

## My Contribution
I developed the player controller, movement mechanics and core game logic.

## Technologies
- Unity
- C#
- Object Oriented Programming (OOP)
